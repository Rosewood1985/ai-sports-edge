SFTP Upload Visual Guide - AI Sports Edge

This guide provides visual instructions for uploading the optimized files to GoDaddy using VS Code's SFTP extension.

⚠️ Important Note

Make sure you have the correct SFTP extension installed. The original extension by liximomo is no longer maintained.

Use the extension maintained by @Natizyskunk instead.

You can install it by running:

./scripts/install-sftp-extension.sh
1

Install the SFTP Extension

Open VS Code and go to the Extensions view (Ctrl+Shift+X or Cmd+Shift+X). Search for "SFTP" and install the extension by @Natizyskunk.

VS Code Extensions view with SFTP extension by @Natizyskunk
2

Save All Files and Restart VS Code

This step is crucial to ensure all configuration changes are applied correctly:

  1. Press Ctrl+K S (Windows/Linux) or Cmd+K S (Mac) to save all files
  2. Close VS Code completely
  3. Reopen VS Code

Critical: Restarting VS Code refreshes the SFTP configuration and ensures the extension works properly. Skipping this step may result in deployment failures.

3

Open VS Code Explorer View

Press Ctrl+Shift+E (Windows/Linux) or Cmd+Shift+E (Mac) to open the Explorer view and navigate to the dist folder in your project.

VS Code Explorer with dist folder highlighted
4

Right-click and Select SFTP: Upload Folder

Right-click on the dist folder and select SFTP: Upload Folder from the context menu.

Right-click context menu with SFTP: Upload Folder option

Note: If you don't see the SFTP: Upload Folder option, make sure you have installed the correct SFTP extension by @Natizyskunk and restarted VS Code.

5

Confirm Upload

A confirmation dialog will appear showing the remote path: /home/q15133yvmhnq/public_html/aisportsedge.app. Click Upload to start the transfer.

SFTP upload confirmation dialog

Note: The upload process may take a few minutes depending on your internet connection speed and the size of the files.

6

Monitor Upload Progress

The VS Code status bar will show the upload progress. Wait for the "Upload complete" notification.

VS Code status bar showing upload progress
7

Verify the Deployment

Open a new incognito/private browser window and visit https://aisportsedge.app.

Check for the following:

  • No reload loops
  • No integrity, MIME, or CSP errors in the browser console (F12 > Console)
  • Firebase authentication works correctly
  • Routing between pages works as expected
  • Spanish language toggle appears and works correctly
8

Test Spanish Language Support

Click the language toggle button in the navigation, verify that text changes to Spanish, and test navigation and functionality in Spanish mode.

Language toggle button in navigation

Troubleshooting

If Files Are Not Uploading

  • Check the SFTP configuration in .vscode-sftp-deploy/.vscode/sftp.json
  • Verify server credentials and connection settings
  • Try restarting VS Code

If X-Frame-Options or Integrity Issues Persist

  • Run the cleanup script again: ./scripts/deploy-without-htaccess.sh
  • Check the HTML files manually for any remaining problematic tags

If Language Toggle Doesn't Appear

  • Verify that language-switcher.js is included in the HTML
  • Check browser console for any JavaScript errors
